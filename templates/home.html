COVID Vax Data API

This is a demo API. The data used comes from Our World In Data. The data was cleaned, normalized, and added to a database in order to create this API. The data used for this API represents global COVID-19 and vaccination data. The data in this API is automatically updated daily.

Routes

HTTP Verb Description Route
GET Get all continent data /continent
GET Get continent data by id /continent/id/<continent_id>
GET Get all country data /country
GET Get country data by id /country/id//<country_id>
GET Get all global COVID-19 data /covid-data
GET Get a country’s COVID-19 data /covid-data/country/<country_name>
GET Get the latest COVID-19 data from all countries /covid-data/latest
GET Get the latest COVID-19 data of a specific country /covid-data/latest/<country_name>
GET Get all daily vaccination data /daily-vax
GET Get the daily vaccination data of a specific country /daily-vax/country/<country_name>
GET Get the latest vaccination data for all countries /daily-vax/latest
GET Get the latest vaccination data for a specific country /daily-vax/latest/<country_name>
GET Get the total number of vaccines administered (at least one dose) per brand globally /total-vax-brand
GET Get the total number of vaccines administered (at least one dose) per brand for a specific country /total-vax-brand/country/<country_name>
GET Get all the vaccine brands available in every country /vax-in-country
GET Get all the vaccine brands available in a specific country /vax-in-country/country/<country_name>
GET Get all the vaccine brands available in every country /vax-in-country/vaccine/<vaccine_name>
GET Get all vaccine data /vaccine
GET Get vaccine data by id /vaccine/id/<vaccine_id>
